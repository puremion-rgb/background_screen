# 🛒 Shopping Shot (쇼핑 샷)

> 쇼핑몰을 돌아다니며 마음에 드는 상품 이미지를 우클릭 한 번으로 저장하고, 나만의 장바구니 리스트를 만들어보는 Chrome 익스텐션 프로젝트입니다.

<br>

## 🌟 주요 특징 (Key Features)

- **⚡ 원클릭 스크랩**: 상품 이미지 위에서 우클릭 메뉴를 통해 즉시 나만의 장바구니에 추가합니다.
- **🔗 스마트 링크 추적**: 다양한 쇼핑몰의 복잡한 HTML 구조를 분석하여 메인 페이지가 아닌 **진짜 상품 상세 페이지 주소**를 정확하게 추적해 저장합니다.
- **📂 편리한 사이드바 UI**: 크롬 내장 `sidePanel` API를 활용하여 웹 쇼핑을 하는 화면 옆에 장바구니를 상시 띄워두고 실시간으로 확인할 수 있습니다.
- **📝 메모 및 🎨 컬러 태그**: 스크랩한 상품에 개인적인 메모(사이즈, 고민 지점 등)를 남기고 4가지 컬러 태그로 카테고리를 분류할 수 있습니다.
- **🔍 실시간 메모 검색**: 저장된 상품이 많아져도 메모 내용을 기반으로 원하는 상품을 실시간으로 필터링합니다.
- **🔄 드래그 앤 드롭 정렬**: 마우스 드래그를 통해 내 마음대로 상품 배치 순서를 바꿀 수 있어 우선순위 비교가 직관적입니다.
- **🛡️ 이미지 유실 방지**: 외부 불펌 방지 스크립트나 링크 만료로 인해 이미지가 깨지는 경우, 기본 대체 아이콘으로 안전하게 렌더링되도록 예외 처리가 되어 있습니다.

<br>

## 🛠️ 기술 스택 (Tech Stack)

- **Architecture**: Chrome Extensions API (Manifest V3)
- **Frontend**: HTML5, CSS3 (Grid Layout, Flexbox, Animation), JavaScript (Vanilla JS, DOM Manipulation)
- **Storage**: `chrome.storage.local` (비동기 데이터 관리 및 상태 동기화)

<br>

## 📁 프로젝트 구조 (Project Structure)

```text
📁 shopping-shot/
 ├── 📄 manifest.json      # 익스텐션 권한 설정 및 서비스 워커/컨텐트 스크립트 등록
 ├── 📄 background.js      # 우클릭 콘텍스트 메뉴 생성 및 사이드바 제어 (Service Worker)
 ├── 📄 content.js         # 웹페이지 내 우클릭 이벤트 감지 및 부모 태그 기반 상세 주소 추출
 ├── 📄 popup.html         # 구글 웹폰트와 모달 창이 포함된 사이드바 화면 구조
 ├── 📄 popup.css          # 호버 애니메이션 및 드래그 앤 드롭 시각 효과가 반영된 스타일시트
 ├── 📄 popup.js           # 데이터 로드/렌더링, 검색, 수정, 정렬 및 토스트 메시지 제어
 └── 🖼️ icon.png           # 익스텐션 공식 로고 및 이미지 로딩 실패 시 대체 이미지
```
